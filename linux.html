Linux installation tutorial (mint)







1) get an installation medium (usb stick)
2) go to the linux mint website
3) download the appropriate version (usually x64). cinnamon is fancy, xfce is lightweight, and MATE is somewhere in between
4) download balenaetcher
5) plug in installation medium, and use etcher to copy the linux mint download onto the usb stick.
6) then, close etcher and restart the laptop.
7) before the boot screen shows up, spam the boot menu key
8) when into the boot menu, choose to boot from the usb stick. when you exit, you should boot into linux mint
9) it is now running off of the usb stick. you now need to install it by pressing on the installation icon.
10) go through all the required steps to install. if you choose to dual boot, the disk will be wiped, and linux will be installed, so be careful
11) done. so easy compared to something like arch.
